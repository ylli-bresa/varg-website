# Work section images

Drop your work/portfolio images here.

- **Formats:** SVG (best for logos), or WebP/PNG/JPG (e.g. 400×400px or 800×800px for sharp 2x).
- **Paths:** Files in this folder are served at `/work/your-filename.ext`.
- **List them:** Add or edit entries in `src/data/work.ts` with `imagePath: "/work/your-filename.ext"`, plus `title`, `slug`, and optional `imageWidth`/`imageHeight`.

Example: if you add `acme-logo.svg` here, in `work.ts` use:
`{ id: "1", slug: "acme", title: "Acme", imagePath: "/work/acme-logo.svg", imageWidth: 400, imageHeight: 400 }`
